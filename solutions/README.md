This folder contains one C++ scaffold for every problem listed in the 10-year Bronze guide.

Important: these are study scaffolds with matched structure, tags, approach notes, and implementation entry points. They are not claimed as fully verified AC submissions for all 117 problems. Use them to write and test solutions problem by problem.
