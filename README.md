# USACO Bronze Static Site

Deploy this folder as the Vercel project root.

Files:

- `index.html`: the static app.
- `solutions/python/`: Python solution scaffolds.
- `solutions/cpp/`: C++ solution scaffolds.
- `USACO_Bronze_10_Years_Guide_EN.md`: English-first study guide with Chinese helper labels.

The app embeds the solution scaffold text for reliable code viewing from `file://` and static hosting. The source files are also stored under `solutions/` for editing and future replacement with verified AC implementations.

Copyright note: the site does not reproduce full USACO problem statements. It provides study-safe descriptions and official statement lookup links.
